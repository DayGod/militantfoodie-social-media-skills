# Skill: Reel Scripting

> Write a complete reel script — hook, scenes, voiceover or text overlay, and CTA — in Mike's voice.

---

## How Claude Uses This Skill

Reels are different from static posts — they move, they have rhythm, they require a hook in the first second. Claude writes reel scripts that are visual-first, punchy, and built around Mike's moody food aesthetic. Every script includes a scene breakdown, timing, and either voiceover copy or text overlay copy (or both).

---

## Instructions

1. **Get the brief:**
   - Dish or concept
   - Reel format (recipe process / technique / day-in-life / opinion / trending audio)
   - Delivery style (voiceover / text overlay / silent with music / talking to camera)
   - Target length (15s / 30s / 60s / 90s)
   - Any audio direction (trending sound, original audio, music mood)
2. **Write a scene-by-scene script:**
   - **Scene 1 (0–2s):** The hook — the single most compelling visual moment, front-loaded
   - **Scenes 2–N:** Story beats, each 3–8 seconds, one idea per scene
   - **Final scene:** CTA — verbal or text overlay
3. **For each scene write:**
   - Shot description (what the camera shows)
   - Duration
   - Voiceover line (if applicable)
   - Text overlay (if applicable)
   - Audio note (music swell, silence, sound design)
4. **Voice rules for voiceover:**
   - Short sentences. Pause between them.
   - Sounds like Mike talking to one person, not presenting to a room
   - Never explains what you can already see on screen
5. **Write a caption brief** to go with the reel (handled fully by the Post Writer skill).

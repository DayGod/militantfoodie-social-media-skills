## Workflow Rules

1. **Voice Profile is always active.** Never write content without it.
2. **Load the skill before the task.** Don't start writing until the relevant skill is loaded.
3. **One CTA per post.** Never stack engagement moves.
4. **Two versions for every caption.** Full story and punchy. Mike picks.
5. **Score before finalising.** If Mike asks for a caption, run post-scorer before calling it done.
6. **No food blogger language.** Ever. See Voice Profile Section 07.
